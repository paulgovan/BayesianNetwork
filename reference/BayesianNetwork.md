# Bayesian Network modeling and analysis.

BayesianNetwork is a Shiny web application for Bayesian Network modeling
and analysis.

## Usage

``` r
BayesianNetwork()
```

## Value

The function returns no value. It launches a shiny web application for
Bayesian Network modeling and analysis.

## See also

<http://paulgovan.github.io/BayesianNetwork/>

## Examples

``` r
if (interactive()) {
  BayesianNetwork()
}
```
